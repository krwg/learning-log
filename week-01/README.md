# About Me — Personal Portfolio Page

A clean, semantic HTML personal portfolio page featuring my bio, tech stack, and contact info. Fully static — no frameworks, no build tools, just pure HTML.

## Quick Start

Get it running locally in 2 seconds:

```bash
git clone https://github.com/krwg/about-me.git
cd about-me
open index.html
``` 
## Features

- Semantic HTML5 (< header>, < nav>, < main>, < section>, < footer>)
- Anchor navigation — smooth jump links to sections
- External link to GitHub profile (opens in new tab)
- mailto: link for direct email contact
- Fully responsive by default (browser-native)
